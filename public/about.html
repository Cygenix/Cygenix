How it works
From connection to completion

Cygenix walks you through every stage of the migration — source, target, AI mapping, validation and execution — in a single browser session.

1

Connect databases

Paste a SQL Server connection string or use an Azure Function with Managed Identity. No firewall changes needed.

2

AI column mapping

Claude AI analyses both schemas and maps every target column automatically — with confidence scores and truncation warnings.

3

Was/Is value translation

Upload a translation table to automatically convert lookup values, status codes and department names during migration.

4

Run validation

Execute pre and post-migration validation scripts from any database table. Any rows returned = data quality issue.

5

Execute & report

Run the migration, monitor step-by-step progress, and export a client-ready Excel report when done.

Platform features
Everything in one place

No stitching together separate tools. Cygenix handles the full lifecycle — from first connection to final report.

🤖

AI column mapping

Claude Haiku maps every target column automatically. Override any row, add fixed SQL values, and get ⚠ truncation warnings when types don't fit.

🔄

One-to-many mapping

Map a single source table into multiple target tables with FK relationships, PK variables and per-card AI mapping — all in one generated SQL script.

✅

Flexible validation

Add any table as a validation source. Pick the SQL column, choose report columns, and run pre/post-migration checks against any database.

💻

SQL Editor

Full-featured SQL workspace with script library, find & replace, results grid, history, Excel export and an AI assistant that writes SQL on request.

📅

Project Planner

Calendar (year/month/week/day), tasks, blockers, phase tracker and activity log — driven by your project completion date. Syncs with Smartsheet and Jira.

⚡

Performance tuning

Configure fetch page size, batch size, concurrency, timeouts and retry behaviour. Presets for Small DB, Large DB, Azure SQL and Cautious mode.

📊

Professional reports

Export client-ready Excel reports with summary, migration results, column mapping, Was/Is rules applied and full execution log.

💬

Ask AI assistant

Project-aware AI with full context of your tasks, jobs, blockers and timeline. Ask anything — it only answers about your migration project.

🔌

Azure native

Azure SQL with Managed Identity — no passwords in connection strings. Deployed on Netlify with an Azure Function backend for secure DB access.
